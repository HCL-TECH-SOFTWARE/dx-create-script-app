**Support for this feature is not available via the HCL Support Center.**

If you find a bug, please help us by reporting it on our issue tracker. This is the **only way** to report code-related problems.

1.  **Check Existing Issues:** Search the [Issues tab](https://github.com/HCL-TECH-SOFTWARE/dx-create-script-app/issues) to see if the bug has already been reported. If it has, please add a comment to the existing issue with any additional information you may have.
